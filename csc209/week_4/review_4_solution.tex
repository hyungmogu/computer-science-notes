            \begin{itemize}
                \item Allows variable to gain access to the object pointed by \texttt{p}
            \end{itemize}
        \end{itemize}
        \item \textbf{Aliases}

        \begin{itemize}
            \item Allows the same memory location can be accessed using different variable names.
